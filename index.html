Summary: Latitude vs. X


The purpose of this project is to analyse how various weather statistics are effected by changes to the latitude. To this we used APIs to extract weather data from capital cities across the world and plotted them against their respective latitude values.

For more indepth analysis and visualisations select the plots for max temp, cloudiness, wind speed or humdiity.
